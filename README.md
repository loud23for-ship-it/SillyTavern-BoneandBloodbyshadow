## 🌙 序言 · 骨与血之理念

**骨为架，血为魂。**  
每一个角色都拥有心跳、秘密、即兴的喜怒哀乐，以及与世界共振的偶然性。  
**“骨与血”** 不是单纯的工具集 —— 而是一个让 RP 长出 **真实血肉** 的生态系统。

- 🦴 **骨架**：稳定的副API架构，不影响主窗口生成，关闭界面仍在后台低语。  
- 🩸 **血液**：自动日记、NPC动态、天气变化、读心氛围、命运突发事件、平行宇宙、世界频率、画廊、情侣空间……每一秒世界都在自主流动。  

> 💡 **设计哲学**：AI 回复的那几秒，你的角色并没有睡着 —— 他们正在写日记，NPC在密谋，天空中飘来一则奇闻，而你只需回头，就能捡起这些“活着的证据”。

## 🧬 高级生态 · 让世界真正“活”过来

### 💬 破墙聊天室 —— 第四面墙粉碎机
> 独立可编辑预设，设定你想要的破墙聊天内容和氛围。**在这里你只是你**，与角色坦诚对话，调侃、剧透、吐槽都可以～  
> 预设内容可**导入/导出**，也可插件内直接编辑。

### 🎛️ 预设与自定义 · 完全由你塑造
- **整体预设**：为日记、情报、平行宇宙等所有模块设定全局AI提示词。  
- **风格预设**：**4种配色主题** + 按钮名称自定义 (◕ᴗ◕✿)  
- **自定义CSS & AI提示词**：快捷调整插件外观，深度美化。  
- 所有预设均支持 **导入/导出/插件内编辑**。

### 🖼️ 画廊 & 情侣空间 & 成就
- 🎨 **画廊**：AI生图管理，所有配图（日记/世界新闻）的仓库。  
- 💌 **情侣空间**：留言板 / 图像存放 / 生图 / 写情书 / **设定纪念日** —— 和你的角色拥有一片秘密花园。  
- 🏆 **成就系统**：收集徽章，记录每一次沉浸瞬间。

### 🌍 世界频率 · 奇闻轶事不间断
> 情报站子模块 —— 随机生成世界事件、都市传说、远方来信…… 让 RP 世界拥有自己的“新闻频道”。

### 🎵 音频体验 · 耳朵也会沉浸
- **MP3模式**：接入音乐API搜索歌曲，或直接使用URL播放。  
- **新内容生成提示音**：日记/情报/命运事件出现时，叮咚一声不错过任何精彩。  
- 完全**不影响主窗口RP生成**，独立音频通道。

### 🛠️ 技术亮点 · 强大且优雅
| 特性 | 说明 |
|------|------|
| 🔄 副API架构 | 所有扩展功能走独立API通道，绝不干扰主对话生成。 |
| 🧩 后台常驻 | 关闭主界面，日记照写、NPC照动、世界频率照播。 |
| 🎨 生图集成 | 为日记内容、世界新闻智能配图（需生图API）。 |
| 📸 导出图片 | 唱片机支持收藏内容 → 自定义字体/背景 → 导出图片。 |
| 🦋 平行空间 | 对某条回复不满意但又不舍得扔？开启平行宇宙，换一条分支继续。 |

## 🖥️ 主界面 · 沉浸式操作台

- 🖼️ **自定义头像 & 背景图**：放上你与角色的头像，三种布局可选。  
- 📊 **聊天统计**：自动记录聊天条数，推测你们共度的总时长 (⏱️ “你们已经聊了23小时16分钟…”)。  
- 🌦️ **天气小组件**：看看今天角色窗外是晴是雨。  
- 💭 **氛围分析（读心术）**：实时洞察角色的内心状态 —— “他似乎有一丝犹豫”。  
- 🎲 **命运之骰**：一个神秘按钮，随时触发随机事件。

## 🎁 更多细节 · 藏起来的浪漫

- 🌟 **唱片机**：长按消息 → 收藏 → 导出为**带自定义字体/背景**的精美图片，或纯Markdown/JSON。  
- 📖 **日记本**：支持设定“每天/每N条对话后自动写日记”，也可手动点按 —— 看着角色慢慢长出内心独白。  
- 👥 **NPC模块**：看看酒馆里的路人、城门口的卫兵此时在做什么，他们也有自己的小剧场。  
- 🔮 **破墙聊天室**：可以预设“吐槽模式/鼓励模式/深夜电台模式”，氛围词条自由编辑。  
- 💝 **情侣空间**：设置纪念日

## 🚀 快速开始 · 让骨血流动

1. **下载插件**  
   复制网页在扩展区域下载~

2. **配置API（可选但推荐）**  
   - 日记/情报/命运盘/观测站 → 需要 LLM API（OpenAI/Claude/本地）。  
   - 生图配图 → 需要图像生成API（如DALL·E、ComfyUI等）。  
   - 音乐搜索 → 需要音乐API或直接放URL。  
   *所有API密钥均在插件设置页面配置，安全隔离。*

3. **开启你的第一个“血液”时刻**  
   - 进入插件面板 → 开启“日记自动生成” → 和角色聊5分钟 → 翻开日记本，你会看到角色写下了属于“他/她”视角的独白。  
   - 点击🦋符号，尝试平行宇宙。  
   - 在情报站里点一下“世界频率”，一则奇闻就会浮现。

4. **自定义你的风格**  
   - 进入风格预设，
   - 上传背景图，调整CSS，导出/导入预设与他人分享。
